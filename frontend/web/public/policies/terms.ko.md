# 이용약관 (v2 초안)

> ⚠️ **본 문서는 Claude 가 작성한 초안 (v2)** 입니다. legal review 보조 결과 반영. 법무 검토 후 최종 게시 예정.

## 1. 서비스 개요

OpenMake LLM (이하 "서비스") 는 자체 호스팅 AI assistant 플랫폼으로, 다중 LLM 모델 라우팅 및 사용자 정의 에이전트/스킬 기능을 제공합니다.

## 2. 계정 및 가입

- 회원가입 시 정확한 이메일과 사용자명을 입력해야 합니다.
- 비밀번호는 8자 이상, 대문자/소문자/숫자/특수문자를 포함해야 합니다.
- 본인의 계정 정보를 안전하게 관리할 책임은 사용자에게 있습니다.
- 타인의 계정 도용, 자동화 도구를 이용한 무단 가입은 금지됩니다.

### 2-a. 14세 미만 가입 (정보통신망법 §31, 개인정보 보호법 §39-3)

- 14세 미만 아동은 **법정대리인의 동의 없이 가입할 수 없습니다**.
- 14세 미만 가입 시 법정대리인 (부모 등) 의 동의 절차가 별도로 진행됩니다 (현재는 운영자에게 별도 문의 — 셀프 동의 흐름은 후속 구현 예정).
- 운영자는 가입 시 사용자가 14세 이상임을 확인할 수 있는 합리적 조치를 취해야 합니다.

## 3. 서비스 이용

### 3.1 허용되는 사용
- 개인 학습, 연구, 업무 목적의 AI 대화
- 사용자 정의 에이전트/스킬 생성 및 사용
- 자체 등록한 외부 LLM API 키로 외부 모델 호출

### 3.2 금지되는 사용

다음 행위는 본 서비스 이용 시 일체 금지됩니다 (정보통신망법 §44-7 의 불법정보 9종 포함).

- 음란물 / 청소년 유해매체물의 생성·유포
- 불법촬영물 (개인의 신체 동의 없는 촬영물) 의 생성·유포
- 명예훼손 / 모욕 / 스토킹 / 협박
- 사기·도박·마약 등 형법 위반 콘텐츠
- 국가기밀 / 미공개 비공식 정보 유포
- 국가보안법 위반 (이적 표현, 반국가단체 찬양 등)
- 저작권·상표권·디자인권·특허권 등 지식재산권 침해
- 타인의 권리 침해 (프라이버시, 초상권, 성명권 등)
- 시스템 무단 침입, 부정 사용, 과도한 트래픽 유발
- 자동화 도구를 이용한 대량 호출 (rate limit 회피)
- 보안 취약점 악용
- 차별/혐오 표현 (성별, 인종, 종교, 출신, 장애 등)
- 가짜뉴스 / 허위정보의 의도적 유포
- 기타 관계 법령 위반 또는 공서양속에 반하는 행위

위반 시 사전 통지 없이 계정이 정지되거나 삭제될 수 있으며, 운영자는 관계 법령에 따라 신고/협조 의무를 이행할 수 있습니다.

## 4. Rate Limit 및 사용량 제한

서비스는 안정적 운영을 위해 다음 제한을 적용합니다.

- 채팅 요청: 시간당/주간 토큰 한도 (`LLM_HOURLY_TOKEN_LIMIT`, `LLM_WEEKLY_TOKEN_LIMIT`)
- API 키 관리, MCP 서버 등록, 스킬 생성 등 관리 작업별 별도 quota
- 데이터 내보내기: 시간당 사용자별 N회 (Tier 별 차등 — 무료/Pro/Enterprise)
- 한도 초과 시 일정 시간 후 자동 해제

## 5. 콘텐츠 소유권 및 저작권

### 5-1. 사용자 콘텐츠
- 사용자가 입력한 prompt, 업로드 파일의 저작권은 **사용자에게 귀속**됩니다.
- 사용자가 생성한 에이전트/스킬의 저작권은 **사용자에게 귀속**되며, 사용자가 명시적으로 공개 (`is_public=true`) 설정한 skill manifest 는 다른 사용자에게 공유될 수 있습니다.
- 사용자 탈퇴 시 본인이 작성한 manifest 는 자동으로 비공개 처리되어 더 이상 공유되지 않습니다 (개인정보 처리방침 §4 참조).

### 5-2. AI 생성 응답
- AI 모델이 생성한 응답의 저작권 귀속은 **명확하지 않은 영역** 이며 사용 시 다음에 유의하시기 바랍니다.
  - 한국 저작권법상 인간 창작물만 보호 대상 — AI 단독 출력의 저작권은 보호받지 않을 수 있습니다.
  - 사용자가 prompt + 편집 등을 통해 창작 기여한 부분은 별도 보호 가능.
- 운영자는 AI 출력의 정확성·완전성·적법성에 대해 보장하지 않습니다.
- 사용자는 AI 출력의 사용 (배포·상업 이용 등) 시 자체 검증 책임이 있습니다.

## 6. 면책 조항

- 본 서비스는 "있는 그대로" 제공되며, 특정 목적에 대한 적합성을 보장하지 않습니다.
- AI 응답의 정확성, 완전성, 적법성에 대해 보장하지 않습니다.
- 외부 LLM 제공자 (Anthropic, OpenAI, Gemini 등) 의 서비스 중단으로 인한 영향은 본 서비스의 책임이 아닙니다.
- 서비스 운영 중단, 데이터 손실, 부정 사용으로 인한 손해에 대해 운영자의 고의/중과실이 없는 한 책임지지 않습니다.

### 6-a. 강행규정 우선
다음 사항은 본 약관과 무관하게 관계 법령 (소비자기본법, 약관규제법, 전자상거래법, 정보통신망법 등) 의 강행규정이 우선합니다.

- 약관 조항이 강행규정에 반하는 경우 그 조항은 무효
- 소비자의 정당한 권리 (청약 철회권, 손해배상 청구권 등) 는 면제되지 않음
- 운영자의 고의/중과실로 인한 손해배상 책임은 면제 불가

## 7. 약관 변경

- 운영자는 법령 또는 운영 정책 변경에 따라 본 약관을 변경할 수 있습니다.
- **중요한 변경** (사용자에게 불리하거나 권리·의무에 중대한 영향을 미치는 변경) 의 경우 변경 효력 발생 **30일 전 이상** 사전 통지합니다 (전자상거래법 §15).
- **경미한 변경** 은 변경 후 즉시 통지합니다.
- 통지 방법: 서비스 첫 진입 시 modal (재동의 prompt) + 이메일 (보조).
- 통지 기간 내 사용자가 반대 의사를 표시하지 않고 서비스를 계속 이용하는 경우 변경에 동의한 것으로 간주합니다. 반대 시 계정 해지 가능.

## 8. 분쟁 해결

- 본 약관은 **대한민국 법령** 에 따라 해석됩니다.
- 분쟁 발생 시 우선 운영자의 고충 처리 부서 (개인정보 처리방침 §11) 에 협의를 신청할 수 있습니다.
- 협의가 이루어지지 않을 경우 다음 기관의 조정을 신청할 수 있습니다.
  - **한국소비자원** 소비자분쟁조정위원회 (ccn.go.kr / 국번없이 1372)
  - **한국인터넷진흥원** 전자거래분쟁조정위원회 (ecmc.or.kr)
- 위 조정이 성립되지 않는 경우 **운영자 소재지를 관할하는 대한민국 법원** 을 1심 관할 법원으로 합니다 (단 소비자 거주지 법원도 관할 가능 — 민사소송법 §8).

## 9. 문의

본 약관 관련 문의는 서비스 관리자에게 연락 바랍니다 (개인정보 처리방침 §11 의 고충 처리 부서 동일).
