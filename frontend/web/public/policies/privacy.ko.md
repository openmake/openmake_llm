# 개인정보 처리방침 (v2 초안)

> ⚠️ **본 문서는 Claude 가 작성한 초안 (v2)** 입니다. legal review 보조 결과의 High 4개 + 일부 Medium 항목 반영. 법무 검토 후 최종 게시 예정. 운영자가 실서비스 배포 전 반드시 법률 자문을 받으세요.

OpenMake LLM (이하 "서비스") 은 사용자의 개인정보를 「개인정보 보호법」 및 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」, GDPR 에 따라 다음과 같이 처리합니다.

## 1. 수집하는 개인정보

회원가입 및 서비스 이용 과정에서 다음 정보를 수집합니다.

| 항목 | 수집 시점 | 필수 여부 |
|------|----------|----------|
| 사용자명 (username) | 회원가입 | 필수 |
| 이메일 주소 | 회원가입 | 필수 |
| 비밀번호 (단방향 해시) | 회원가입 | 필수 |
| IP 주소, User-Agent | 회원가입·로그인·동의 시점 | 자동 수집 |
| 인증 쿠키 (HttpOnly JWT auth_token) | 로그인 후 | 자동 생성 |
| 대화 내용, 업로드 파일 | 서비스 이용 시 | 사용자 입력 |
| 외부 LLM API 키 (사용자 등록) | 사용자 설정 시 | 선택 |

### 1-a. 자동 수집 장치 (쿠키) 및 거부 방법

- 본 서비스는 인증 유지를 위해 HttpOnly 쿠키 (`auth_token`) 를 사용합니다.
- 분석 / 광고 / tracking 쿠키는 사용하지 않습니다.
- 거부 방법: 브라우저 설정에서 쿠키 차단 가능. 단 인증 쿠키 차단 시 로그인 사용 불가.

## 2. 개인정보 처리 목적

- **계정 식별 및 인증**: 사용자명, 이메일, 비밀번호 해시, 인증 쿠키
- **서비스 제공**: 대화 세션 관리, 모델 라우팅, 사용자별 설정
- **보안 및 감사**: IP/User-Agent, `audit_logs` (관리자 작업 추적), `consent_logs` (GDPR Article 7 / 개보법 §17 동의 demonstrability)
- **부정 사용 방지**: 비정상 트래픽 차단, rate limiting

## 3. 보유 및 이용 기간

- **계정 활성 중**: 사용자가 직접 삭제 요청하기 전까지 보관 — 단 아래 정보통신망법 분리보관 정책 적용
- **1년 미사용 분리보관 (정보통신망법 §29)**: 마지막 로그인 후 1년 경과 시 별도 저장소로 분리 (운영 정책으로 후속 구현 예정)
- **계정 삭제 시**: 아래 §4 "사용자 탈퇴 시 데이터 처리" 정책 적용
- **audit_logs / message_feedback**: 사용자 식별자 익명화 (`user_id → NULL`), 본문은 5년 보존 (감사 추적 의무)
- **consent_logs**: 사용자 탈퇴 시 함께 삭제 (CASCADE). PII (IP/UA) 는 90일 후 자동 익명화 (Phase C Fix 9)

## 4. 사용자 탈퇴 시 데이터 처리

서비스의 데이터 모델은 사용자 탈퇴 시 데이터를 세 카테고리로 분류 처리합니다.

### (A) 즉시 삭제되는 데이터 (CASCADE)
- 커스텀 에이전트 (custom_agents)
- 개인 스킬 (agent_skills)
- 대화 세션 (conversation_sessions)
- 사용자 메모리 (user_memories)
- 외부 API 키, OAuth 연결 (external_connections, user_api_keys)
- MCP 서버 인스턴스/등록
- 푸시 구독 (push_subscriptions)
- 동의 이력 (consent_logs)

### (B) 작성자 익명화 후 보존 (SET NULL)
- **감사 로그 (audit_logs)** — 관리자 작업 추적 의무
- **메시지 피드백 (message_feedback)** — 모델 평가 데이터
- **Skill manifest (skill_manifests)** — manifest 자체는 보존하되, `created_by` 가 NULL 로 설정되고 `is_public` 이 자동 false 처리되어 **다른 사용자에게 노출되지 않습니다** (Phase A Fix 1 보호 정책)

### (C) 일부 데이터는 별도 정리 후 삭제
사용자가 다른 사용자에게 남긴 콘텐츠 (리뷰, 마켓플레이스 등록 등) 는 다른 사용자 활동과 연관되어 있어 별도 정리 후 삭제 가능합니다. 정리되지 않은 데이터가 있을 경우 계정 삭제가 일시 차단될 수 있으며, 관리자가 사전 정리 안내를 드립니다.

## 5. 정보주체의 권리

사용자는 다음 권리를 행사할 수 있습니다 (GDPR Article 15-22 + 개인정보 보호법 §35-39).

### 행사 방법
- **온라인**: 설정 페이지 (`/settings.html`) 에서 직접 행사 가능 항목
  - 데이터 내보내기 (열람 + 이동) — "데이터 내보내기" 버튼
  - 동의 철회 — "동의 관리" 섹션
  - 이메일/사용자명 수정 — 계정 설정
  - 계정 삭제 요청 — 별도 문의 (현재 admin 처리, 본인 셀프 삭제 후속 구현)
- **이메일 문의**: §11 고충 처리 부서 또는 §10 개인정보 보호책임자 (CPO)

### 권리 일람
- **열람권 (Article 15 / §35)**: 보유 중인 개인정보 조회
- **정정·삭제권 (Article 16-17 / §36)**: 잘못된 정보 수정 또는 삭제
- **처리 정지권 (Article 18 / §37)**: 별도 문의
- **이동권 (Article 20)**: 데이터 내보내기 JSON
- **반대권 (Article 21 / §38)**: 별도 문의
- **자동화 의사결정 거부 (Article 22)**: 현재 자동화 의사결정 미사용

## 6. 동의 (GDPR Article 7 / 개인정보 보호법 §15)

회원가입 시 본 정책 및 이용약관에 대한 명시적 동의를 수집합니다.

- **동의 거부권**: 약관 / 처리방침 동의는 회원가입 필수 — **거부 시 회원가입 불가** (다만 게스트 모드는 별도 제공 가능)
- **거부 시 불이익**: 회원가입을 통한 개인화 기능 (대화 세션 저장, 스킬 등록 등) 사용 불가. 게스트 모드는 일부 기능 제한.
- **동의 이력 보관 정보**: 동의 시점, 정책 버전, locale, IP 주소, User-Agent
- **철회**: 설정 페이지 "동의 관리" 섹션에서 언제든 가능. 철회 후 재로그인 시 재동의 prompt.

## 7. 제3자 제공 및 처리위탁

### 7-1. 제3자 제공 (개보법 §17)
원칙적으로 제3자에게 개인정보를 제공하지 않습니다. 단 다음 예외:
- **법적 요구**: 수사기관/법원의 적법한 영장 또는 명령

### 7-2. 처리위탁 (개보법 §26)
서비스 운영을 위해 다음 처리 업무를 위탁합니다 (사용자 동의 시점에 위탁 인지).

| 수탁자 | 위탁 업무 | 위탁 정보 |
|--------|----------|----------|
| 외부 LLM 제공자 (Anthropic, OpenAI, Gemini 등) | LLM 호출 처리 | 사용자가 본인 API 키 등록한 경우에 한해, 사용자 입력 prompt 가 해당 제공자에게 전송됨 |
| 셀프 호스팅 vLLM/LiteLLM | LLM 호출 처리 (기본 모델) | 사용자 입력 prompt — 자체 인프라 내에서 처리, 외부 전송 없음 |

외부 LLM 제공자 사용 시 해당 제공자의 개인정보 처리방침을 함께 검토하시기 바랍니다.

## 8. 보안

- 비밀번호: bcrypt 단방향 해시 저장
- 세션: HttpOnly JWT 쿠키 (SameSite, Secure 정책 — production 배포 시 HTTPS 의무)
- 외부 API 키: AES-256-GCM 암호화 저장 (`token-crypto.ts`)
- HTTPS: production 환경에서 의무 (Caddy / Cloudflare Tunnel 등)
- Rate Limiting: IP/사용자별 호출 한도 적용

## 9. 정책 변경 시 통지

본 정책의 중요한 변경 (수집 항목 / 처리 목적 / 보유 기간 등 정보주체 권리에 영향 미치는 변경) 시:

- **통지 방법**: 서비스 첫 진입 시 modal (재동의 prompt) + 이메일 (보조)
- **통지 시점**: 변경 효력 발생 **30일 전 이상** 통지 (전자상거래법 §15)
- **반대 의사 표시**: 통지 기간 내 반대 시 계정 해지 또는 변경 전 버전 유지 (별도 협의)
