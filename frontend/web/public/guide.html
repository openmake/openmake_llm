📖 OpenMake.Ai 사용자 가이드

버전 1.5.2 · 마지막 업데이트: 2026-02-14

🎯 소개

OpenMake.Ai은 다양한 AI 모델을 활용하여 대화, 코드 작성, 문서 분석, 번역 등 다양한 작업을 수행할 수 있는 올인원 AI 어시스턴트입니다.

✏️ 프롬프트 모드 (12종)

각 모드는 특정 작업에 최적화된 시스템 프롬프트를 사용합니다. 질문 내용에 따라 자동으로 최적 모드가 선택됩니다.

🟢 기본 모드

🤝 Assistant 기본

일반 대화, 질문 답변, 정보 제공

🧠 Reasoning

복잡한 문제 해결, 수학적/논리적 분석

💻 개발 모드

👨‍💻 Coder

프로그래밍, 디버깅, API 개발

🔍 Reviewer

코드 품질 분석, 보안 취약점 검토

🏗️ Generator

프로젝트 스캐폴딩, 보일러플레이트 생성

🛡️ Security

보안 취약점 분석, 방어 전략 수립

📝 콘텐츠 모드

✍️ Writer

블로그, 이메일, 보고서 작성

📚 Explainer

복잡한 개념을 쉽게 설명

🌐 Translator

다국어 번역, 문화적 맥락 해설

🔬 분석 모드

🔎 Researcher

리서치, 데이터 조사, 트렌드 분석

💼 Consultant

비즈니스 전략, 문제 해결

🤖 Agent

외부 도구 호출, 자동화 작업

🔧 MCP 도구

외부 서비스와 연동하여 LLM의 능력을 확장합니다.

도구 기능
🔍 웹 검색 Google/DuckDuckGo/Naver 검색
🐙 GitHub 레포지토리 검색, 이슈 관리, 코드 조회
📊 Exa Search 학술/기술 문서 검색
📄 PDF 분석 문서 요약 및 Q&A
🖼️ 이미지 분석 Vision 기반 이미지 인식

📄 문서 분석

지원 형식

  • PDF, DOCX, TXT, MD
  • 이미지 (PNG, JPG) - OCR 지원
  • 엑셀 (XLSX) - 데이터 추출

사용 방법

  1. 채팅 입력창의 📎 버튼 클릭
  2. 파일 업로드 (최대 100MB)
  3. "이 문서 요약해줘" 또는 "3페이지 내용 설명해줘"

💡 팁 & 트릭

✅ 효과적인 프롬프트 작성

좋은 예시: "TypeScript로 사용자 인증 API를 작성해줘. JWT 토큰을 사용하고 에러 처리를 포함해줘."

나쁜 예시: "API 만들어줘"

⚠️ 주의사항

AI의 지식은 2024년 12월까지의 데이터 기반입니다. 최신 정보는 웹 검색 기능을 활용하세요.

❓ FAQ

Q: 어떤 모델을 사용하나요?

기본적으로 Gemini 3 Flash/Pro를 사용하며, 설정에서 변경할 수 있습니다.

Q: 대화 내용은 저장되나요?

로그인 사용자의 대화는 서버에 저장되며 히스토리 메뉴에서 확인할 수 있습니다.

Q: 파일 업로드 용량 제한은?

최대 100MB입니다.

Q: 프롬프트 모드를 수동으로 변경할 수 있나요?

네, 사이드바의 프롬프트 모드 섹션에서 원하는 모드를 클릭하세요.
